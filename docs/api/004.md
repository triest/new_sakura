## Связать сопровождающего с ребенком

Запрос:

    Метод: 
        merge_child_concomitants
    Параметры:
        child_id - id ребенка (обязательно)
        concomitant_id - id сопровождающего (обязательно)

Пример ответа:

    array:2 [▼
      "error" => 0
      "status" => "success"
    ]
